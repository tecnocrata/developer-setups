# Post Installation Instructions

## Configure Github Tokens

Generate your keys and copy the pub key

```
ssh-keygen -t rsa -b 4096 -C "<<your email>>"
cat ~/.ssh/id_rsa.pub
```

Goto to Github and SSH Keys section and place it there
